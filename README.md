# ICP Dev Test - RoR

RESTfull API for claim management

## Getting Started

```
$ git clone git@github.com:BabenkoOleg/icp-dev-test.git
$ cd ./icp-dev-test
```

### Prerequisites

The application uses ruby 2.5.3

### Installing dependencies and running

```
$ bundle install
$ rails s
```

## Running the tests

```
bundle exec rspec spec
```
